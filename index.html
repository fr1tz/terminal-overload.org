How to support the project?

TOL is still mostly a one-man project and needs all the help it can get. Please consider taking a few seconds and doing as many of the following tasks as you can.


If you're on Twitter:

  • - Follow Terminal Overload on Twitter to be informed about events and new releases.


If you have some spare cash:

  • - Register your player name (costs 5 bucks)
  • - Become a patron of TOL's main developer (donate whatever amount you want per month)
