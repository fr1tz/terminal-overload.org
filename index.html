It is the continuation of ROTC: Ethernet and is currently in an early stage of development.

How to support the project?

TOL is still mostly a one-man project and needs all the help it can get. The easiest way you can help is by taking a few seconds and doing as many of the following tasks as you can.


If you're on Twitter:

  • - Follow Terminal Overload to be informed about events and new releases: Follow @_NOTC_
  • - Tweet about the project: Tweet


If you're on Google Plus:

  • - Share on Goole Plus:


If you have some more time:

  • - Spread the word anywhere you can (Forums, Subreddits, other social media).


And if you have some spare cash:

  • - Register your player name (costs 5 bucks)
  • - Become a patron of TOL's main developer (donate whatever amount you want per month)
